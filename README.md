# Tic Tac Toe (Java Swing)

This is a simple Tic Tac Toe game implemented using Java Swing. It provides a graphical interface to play Tic Tac Toe with another player. 
The game tracks the score of the players and displays the winner or a draw at the end of each game.

## Features

- Two-player game mode (Player X and Player O).
- Score tracking for each player.
- Displays the winner after each game.
- Option to reset the game or play again after a game ends.
- Stylish UI with color-coded components.

## Technologies

- **Java 8+**: Java programming language used to build the game.
- **Java Swing**: Used for creating the graphical user interface (GUI).
- **GridLayout**: Used to organize the Tic Tac Toe board.

## Setup Instructions

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/your-username/tic-tac-toe-java.git

2. Navigate to the project directory:
  cd tic-tac-toe-java
3. Compile the Java files:
  javac *.java
4. Run the game:
  java Main

## Gameplay
- Click the buttons on the board to place your mark (X or O).
- Player X always starts first.
- The game will announce the winner or a draw after each round.
- You can reset the game or start a new round using the respective buttons.
